@synapxlab/version-checker

Open the browser console to see the version check results.

Example 1: Basic usage

checkVersion({
  currentVersion: '2.4.0',
  packageName: '@synapxlab/cookie-consent'
});

Example 2: Custom callbacks

checkVersion({
  currentVersion: '1.0.0',
  packageName: '@synapxlab/cookie-consent',
  delay: 2000,
  chance: 1, // Always execute (for demo)
  onUpdate: (latest, current) => {
    alert(`Update: ${current} â†’ ${latest}`);
  }
});
Note: This example uses chance: 1 to always execute. In production, use the default 0.10 for 10% sampling.

Example 3: Silent mode with custom UI

checkVersion({
  currentVersion: '2.4.0',
  packageName: '@synapxlab/cookie-consent',
  silent: true,
  onUpdate: (latest) => {
    document.body.innerHTML += 
      `<div class="banner">v${latest} available!</div>`;
  }
});
